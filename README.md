# 2048 Game in Rust

> This is a simple implementation of the 2048 game in Rust. It is a work in progress.

## How to play
```bash
cargo run
```

## Direction keys
- Up: `w`
- Down: `s`
- Left: `a`
- Right: `d`

## How to test
```bash
cargo test
```
